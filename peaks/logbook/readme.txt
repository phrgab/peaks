Current logbook maker functionalities


The logbook maker currently works for the following systems:
St Andrews Phoibos (ARPES system)
St Andrews MBS (Spin-ARPES system)
Diamond i05-HR
Diamond i05-nano
MAX IV Bloch
Elettra APE
SOLEIL CASSIOPEE


Instructions on how to use the logbook maker are in the 'Logbook maker' Jupyter notebook file saved in the examples/logbook folder
